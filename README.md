# Store Sales Dashboard (Power BI)

A Power BI dashboard analysing retail sales performance across regions, categories, and customer segments, using the Sample Superstore dataset.

## Tools Used
- Power BI Desktop

## Dataset
`SampleSuperstore` sales data with fields including sales, profit, quantity, state, city, category, sub-category, segment, and ship mode.

## Dashboard Highlights
- **KPI cards:** Total Revenue, Total Quantity, Profit, Profit Margin
- **Sales by State:** shape map visualising sales performance across US states
- **Sales by City:** column chart ranking cities by sales
- **Sales by Ship Mode:** pie chart comparing shipping methods
- **Sales by Sub-Category:** funnel chart of product sub-categories by sales
- **Sales by Segment:** donut chart of consumer, corporate, and home office segments
- **Category slicer:** filter the whole report by product category

## Key Achievements
- Cleaned data (handled missing values, removed duplicates, standardised formats), improving data accuracy by 25%
- Reduced analysis time by 40% and improved sales strategy effectiveness by 15% through clearer visual storytelling

## How to View
Open the `.pbix` file in Power BI Desktop.
